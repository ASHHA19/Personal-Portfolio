# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/ASHHA19/Personal-Portfolio.git
```

Go to project directory

```bash
cd Personal-Portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
