# Personal Portfolio

This is a Next.js portfolio website built for ASHHA G.

## What’s included

- Responsive portfolio landing page
- Looping background video
- Resume download button linked to `public/resume.pdf`
- Contact information and social links
- Sections for about, skills, projects, experience, certifications, and contact

## Local setup

```bash
cd "c:\Users\VARUN COMPUTERS\Desktop\AR Portfolio"
corepack pnpm install
corepack pnpm dev
```

Then open `http://localhost:3000` in your browser.
